# House Price Prediction (Machine Learning Project)

An end-to-end **Machine Learning project** that predicts price of house
🚀 **Live Demo:** 
https://house-price-prediction-ml-poject.streamlit.app/
---



## 🎯 Features

* 📊 Data preprocessing and analysis
* 🤖 Machine Learning model training
* 📈 Prediction system
* 🌐 Interactive web app using Streamlit
* 🚀 Deployed and accessible online

---

## 🧠 Tech Stack

* **Python**
* **NumPy, Pandas, xgboost**
* **Scikit-learn**
* **Streamlit**
* **Pickle (Model Saving)**

---

---

## ⚙️ How It Works

1. Load sonar dataset
2. Preprocess data
3. Train classification model
4. Save trained model (`.pkl`)
5. Use Streamlit UI for prediction

---

## 🚀 Run Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Just-Binod/house_price_prediction.git
cd house_price_prediction

```

---

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Run the app

```bash
streamlit run app.py
```
